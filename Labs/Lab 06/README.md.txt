# Lab 6: Regression & Classification

## Objective
The objective of this lab was to understand the difference between regression and classification problems in machine learning and to apply basic models for both types.

## Regression vs Classification

### Regression
Regression is used when the output is a continuous value.
- Example: Predicting house prices
- Output is a numerical value

### Classification
Classification is used when the output is a category or class.
- Example: Spam detection (spam or not spam)
- Output is a label or class

## Models Used
In this lab, I explored basic machine learning models such as:
- Linear Regression (for regression problems)
- Logistic Regression (for classification problems)
- Other simple classification models

## Key Differences Learned
- Regression predicts continuous values
- Classification predicts categories
- Different evaluation metrics are used for each type

## Importance in Machine Learning
Understanding the difference between regression and classification is essential because it helps in choosing the correct model for a given problem.

## What I Learned
- How to identify whether a problem is regression or classification
- The basic models used for each type
- How model selection depends on the problem type

## Conclusion
This lab helped me understand the fundamental difference between regression and classification, which is essential for building correct machine learning models.
