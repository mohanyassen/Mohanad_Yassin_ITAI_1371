# Lab 8: Overfitting, Underfitting, and Regularization

## Objective
The objective of this lab was to understand common problems in machine learning models such as overfitting and underfitting, and how regularization helps improve model performance.

## Overfitting
Overfitting happens when a model learns the training data too well, including noise and unnecessary details. This causes poor performance on new, unseen data.

## Underfitting
Underfitting happens when a model is too simple and cannot capture the underlying patterns in the data. This leads to poor performance on both training and test data.

## Regularization
Regularization is a technique used to reduce overfitting by adding a penalty to complex models. It helps improve generalization on unseen data.

Common regularization techniques include:
- L1 Regularization (Lasso)
- L2 Regularization (Ridge)

## Why This Is Important
Understanding overfitting and underfitting is important because it helps in choosing the right model complexity and improving real-world performance.

## What I Learned
- How to identify overfitting and underfitting
- How model complexity affects performance
- How regularization improves generalization
- Why simpler models can sometimes perform better
