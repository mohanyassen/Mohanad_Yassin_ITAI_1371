# Lab 4: Working with Data & Exploratory Data Analysis (EDA)

## Objective
The objective of this lab was to learn how to work with real datasets and perform Exploratory Data Analysis (EDA) to understand the structure, patterns, and quality of data.

## Working with Data
In this lab, I practiced loading datasets using Python libraries such as Pandas. I learned how to inspect data using functions like:
- head()
- info()
- describe()

These tools help understand the structure and basic statistics of a dataset.

## Exploratory Data Analysis (EDA)
EDA is the process of analyzing data to discover patterns, detect anomalies, and understand relationships between variables.

In this lab, I learned how to:
- Identify missing values
- Understand data distribution
- Analyze feature relationships
- Use summary statistics

## Importance of EDA
EDA is an important step in machine learning because it helps:
- Improve data understanding before modeling
- Detect errors or inconsistencies in data
- Guide feature selection and preprocessing decisions

## What I Learned
- How to explore datasets using Python
- The importance of understanding data before building models
- How EDA improves machine learning model performance

## Conclusion
This lab helped me develop essential data analysis skills that are required before applying machine learning algorithms.
