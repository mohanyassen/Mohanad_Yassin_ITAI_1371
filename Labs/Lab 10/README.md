# Lab 10: Unsupervised Learning

## Objective
The objective of this lab was to understand unsupervised learning techniques and how they can be used to find patterns in data without labeled outputs.

## What is Unsupervised Learning?
Unsupervised learning is a type of machine learning where the data does not contain labeled outputs. The model tries to find hidden patterns, structures, or groupings within the data.

## Common Techniques

### 1. Clustering
Clustering is used to group similar data points together.
- Example: K-Means Clustering

### 2. Dimensionality Reduction
This technique reduces the number of features while keeping important information.
- Example: PCA (Principal Component Analysis)

## Key Differences from Supervised Learning
- No labeled output data is used
- The model learns patterns on its own
- Results are often used for grouping or exploration

## Applications
- Customer segmentation
- Anomaly detection
- Data compression
- Pattern discovery
