# Lab 3: Machine Learning Workflow & Types of Learning

## Objective
The objective of this lab was to understand the general workflow of a machine learning project and the different types of learning used in machine learning.

## Machine Learning Workflow
In this lab, I learned the standard steps of a machine learning pipeline:
1. Data Collection
2. Data Preprocessing
3. Splitting Data into Training and Testing Sets
4. Model Selection
5. Model Training
6. Model Evaluation
7. Prediction

Understanding this workflow is important because it ensures that machine learning models are built in a structured and reliable way.

## Types of Machine Learning
I also learned about the main types of machine learning:

### 1. Supervised Learning
- The model is trained using labeled data
- Example: Classification and Regression

### 2. Unsupervised Learning
- The model works with unlabeled data
- Example: Clustering and Dimensionality Reduction

### 3. Reinforcement Learning
- The model learns by interacting with an environment and receiving rewards or penalties

## What I Learned
- The importance of following a structured ML workflow
- The difference between supervised and unsupervised learning
- How reinforcement learning differs from traditional datasets

## Conclusion
This lab helped me understand the foundation of machine learning and how different learning types are applied depending on the problem.
