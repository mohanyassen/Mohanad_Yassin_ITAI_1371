# Lab 11: Hyperparameter Tuning & AutoML

## Objective
The objective of this lab was to understand how to improve machine learning models by tuning hyperparameters and using automated machine learning (AutoML) techniques.

## What are Hyperparameters?
Hyperparameters are settings in a machine learning model that are not learned from data but are set before training. Examples include:
- Number of neighbors in KNN
- Maximum depth in Decision Trees
- Number of estimators in Random Forest

## Hyperparameter Tuning
Hyperparameter tuning is the process of finding the best combination of parameters to improve model performance.

Common techniques include:
- Grid Search
- Random Search

These methods help improve model accuracy and generalization.

## What is AutoML?
AutoML (Automated Machine Learning) is a process that automatically tests different models and parameters to find the best performing model with minimal human effort.

## Why This Is Important
Hyperparameter tuning improves model performance and ensures better accuracy and reliability in real-world applications.

## What I Learned

- The concept of AutoML and automation in machine learning
- How tuning improves model accuracy and generalization
