# Lab 7: Evaluating Machine Learning Models

## Objective
The objective of this lab was to learn how to evaluate machine learning models using different performance metrics and understand how to interpret model results.

## Importance of Model Evaluation
After training a machine learning model, it is important to evaluate how well it performs on unseen data. Evaluation helps us understand whether the model is reliable and suitable for real-world use.

## Evaluation Metrics

### 1. Accuracy
Accuracy measures how many predictions were correct out of all predictions made.

### 2. Precision
Precision measures how many of the predicted positive cases were actually correct.

### 3. Recall
Recall measures how many actual positive cases were correctly identified by the model.

### 4. F1-Score
F1-score is the balance between precision and recall.

## Why Multiple Metrics Are Important
Accuracy alone is not always enough, especially in imbalanced datasets. Metrics like precision and recall provide better insight into model performance.

## What I Learned
- How to evaluate machine learning models using multiple metrics
- The importance of precision and recall in classification problems
- How F1-score balances model performance
- Why accuracy can be misleading in some cases
