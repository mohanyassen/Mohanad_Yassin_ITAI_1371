# Midterm Project: Heart Disease Prediction

## Objective
The goal of this project was to build a machine learning model that predicts whether a patient has heart disease based on medical features.

## Problem Type
This is a binary classification problem:
- 0 = No heart disease
- 1 = Heart disease present

## Dataset
The dataset contains patient health information such as age, cholesterol levels, blood pressure, and other medical indicators.

## Approach
- Loaded and explored the dataset
- Preprocessed the data (handling missing values if needed)
- Split data into training and testing sets
- Trained machine learning models to classify heart disease risk

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest (if used in your notebook)
- KNN (if used)

## Results
The models were evaluated using accuracy and classification metrics. The best-performing model was selected based on performance.
